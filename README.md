# Currency-Converter-using-Html-Css-and-JS
💱 Currency Converter

A simple Currency Converter Web Application built using HTML, CSS, and JavaScript.
It allows users to convert currencies in real-time using an exchange rate API and dynamically updates country flags based on the selected currency.

📌 Features

🌍 Convert between different international currencies

🔄 Real-time exchange rates using an API

🚩 Dynamic country flag updates

🧮 Input any amount to convert

🎨 Simple and responsive UI

⚡ Fast and lightweight frontend project

🛠️ Technologies Used

HTML5 – Structure of the web page

CSS3 – Styling and layout

JavaScript (ES6) – Logic and API handling

Currency API – For live exchange rates

Flags API – For displaying country flags

Font Awesome – Icons

⚙️ How It Works

User enters an amount.

Selects From Currency and To Currency.

Clicks Get Exchange Rate.

The application:

Fetches live data from the currency API

Calculates the converted amount

Displays the result instantly.

Example:

1 USD = 83.2 INR
🔌 API Used
Currency Exchange API
https://cdn.jsdelivr.net/npm/@fawazahmed0/currency-api

Provides latest exchange rates for multiple currencies.
